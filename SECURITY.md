# Security Policy

## Supported Project

This repository is maintained as a student game development project. Security updates will be applied to the current version of the project.

## Reporting a Vulnerability

If you discover a security issue in this repository, please report it privately to the repository owner instead of creating a public GitHub issue.

When reporting a vulnerability, please include:

- A short description of the issue
- Steps to reproduce the issue
- The possible impact
- Any suggested fix, if available

## Security Measures

This repository uses GitHub security features to improve project security, including:

- Dependency Graph
- Dependabot Alerts
- Dependabot Security Updates

These features help identify vulnerable dependencies and support faster security patching.

## Responsible Disclosure

Please do not publicly disclose security issues until they have been reviewed and fixed.
